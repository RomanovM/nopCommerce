What is the process of migration?

Your new nopCommerce store can be up and running in just a few hours. Migration of entities via Cart2Cart is performed automatically and doesn’t require any tech skills. To launch nopCommerce data import follow the steps below:

  1. Create a Cart2Cart account

    Register a new Cart2Cart account or sign up with your Facebook or Google+ account. After the registration process is finished, click Start Demo migration to migrate to your eCommerce data by yourself or Get migration package to get Cart2Cart technical engineers to help you.

  2. Setup your source cart

    From a drop-down list choose your current shopping cart and provide its URL.

  3. Setup your target cart

    Select nopCommerce as a Target cart and provide its URL. After that, find an API plugin in your Admin area (Configuration → Plugins→ All plugins and themes) and download it.

    Note: as this plugin is not inbuilt, first you need to install it from the marketplace. In order to install the API plugin:

    • Upload it to plugins folder in your nopCommerce directory.
    • Restart your app (or click Reload list of plugins button).
    • In the list of plugins find the newly installed plugin.
    • Click on the Install link to install the plugin.
    • The plugin is displayed in the Plugins windows (Configuration → Plugins → Local Plugins).

    After the plugin is installed you will see the new API field in your Admin area. Tick Enable API and then click Save in the right upper corner.

    Go to API → Clients → Add New Client

    • Add Name: create your own Client Name
    • Client ID and Client Secret will be automatically generated. Cart2Cart will ask you to provide your Client ID and Client Secret - copy and paste them to Migration Wizard
    • Callback url: https://www.getpostman.com/oauth2/callback
    • Tick Is Active
    • Click Save in the right upper corner

  4. Choose entities

    Pick up particular entities you want to migrate to nopCommerce or choose Select all.

  5. Choose additional options

    After you select entities you want to migrate to nopCommerce with Cart2Cart, you can optionally pick up some additional options to extend your migration possibilities. In the end of each line with an option name you can find the detailed description and video explanation how the particular option works. Choose what you need and proceed with the migration process.

  6. Start free demo migration

    Once the entities and additional options are chosen, you can start your Free Demo migration to nopCommerce by clicking a Start Free Demo button.

    After Demo migration is set up, you’ll see progress bars showing all the process. During Demo migration, a limited number of entities will be automatically moved to your live nopCommerce store (or Cart2Cart Test store) within 30 minutes, giving an ability to test the service and quality of data migration.

  7. Perform full data migration

    After evaluating Demo results, set up Full migration to nopCommerce by clicking Start migration button. Once the process is over, you will receive a notification email.

    Note that the migration process will not affect your current store performance - it will be available for shoppers.
